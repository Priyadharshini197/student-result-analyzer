# Student Result Analyzer with Visual Reports

## Overview

A Python-based mini project that analyzes student performance from a CSV dataset and generates insights such as grades, topper, pass/fail statistics, and visual reports.

This project demonstrates the power of Python in **data analysis and visualization** using real-world scenarios.

## Features

* Read student data from CSV file
* Calculate total, average, and grades
* Identify topper and rank students
* Detect failed students
* Generate bar chart (subject-wise averages)
* Generate pie chart (pass vs fail)
* Export detailed PDF report



## Technologies Used

* Python
* pandas
* matplotlib
* FPDF



##  Installation

```bash
pip install pandas matplotlib fpdf
```


##  How to Run

```bash
python main.py
```
